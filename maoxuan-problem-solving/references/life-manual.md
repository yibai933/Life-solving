# 从问题到胜利：毛选式人生问题解决手册

这份手册不是摘录，也不是历史复述，而是把《毛泽东选集》中最有行动价值的方法抽象成普通人可使用的 16 个思维模型。它服务于五类问题：战略、决策、执行、反内耗、长期人生方向。

底层原则只有一句：不要在想象中解决问题，要在真实情况、主要矛盾、具体行动和反复复盘中解决问题。

## 使用总则

1. 先看事实，不先看情绪。
2. 先找主要矛盾，不平均用力。
3. 先定阶段，不幻想一步到位。
4. 先做可验证动作，不沉迷完美方案。
5. 先取得主动权，不被外部节奏拖着走。
6. 先组织力量，不把人生过成单兵硬扛。
7. 先处理内部矛盾，不轻易把普通分歧敌我化。

## 模型汇总
| 模型            | 用来解决什么问题        |
| ------------- | --------------- |
| 1. 调查研究模型     | 不知道真实问题是什么时     |
| 2. 主要矛盾模型     | 事情太乱、抓不住重点时     |
| 3. 实践-认识循环模型  | 想太多、行动少、反复纠结时   |
| 4. 持久战模型      | 面对长期困难、焦虑速成时    |
| 5. 战略退却模型     | 暂时失败、要不要放弃时     |
| 6. 集中优势兵力模型   | 精力分散、什么都想做时     |
| 7. 阶段论模型      | 大目标太远、不知道先做什么时  |
| 8. 主动权模型      | 被环境、人际、工作牵着走时   |
| 9. 统一战线模型     | 需要合作、借力、争取支持时   |
| 10. 政策边界模型    | 做事容易过火或摇摆时      |
| 11. 领导方法模型    | 带团队、推进项目、影响他人时  |
| 12. 报告制度模型    | 信息混乱、反馈断裂、执行失控时 |
| 13. 批评与自我批评模型 | 处理内耗、自责、关系摩擦时   |
| 14. 人民内部矛盾模型  | 判断冲突性质，避免敌我化时   |
| 15. 组织起来模型    | 一个人扛不住，需要系统化时   |
| 16. 艰苦奋斗模型    | 资源少、起点低、长期积累时   |
## 模型 1：调查研究模型

核心原理：没有真实调查，就没有可靠判断。许多痛苦不是因为问题太难，而是因为人用情绪、二手信息和脑内剧场代替事实。

操作方法：
1. 写下问题的事实版：发生了什么，发生几次，谁参与，结果是什么。
2. 写下推测版：我以为、我担心、我猜测的部分是什么。
3. 找到缺口：现在最缺的是数据、反馈、对方真实想法、市场反应，还是自己的实际能力水平。
4. 做一个小调查：问 3 个人、试 1 次、记录 7 天、复盘 1 个项目、整理 10 条真实反馈。
5. 用事实重写问题。

误区：
- 把查资料当调查。资料是外部信息，调查要能解释自己的具体处境。
- 把焦虑当证据。焦虑只说明你在意，不说明判断正确。
- 永远调查不行动。调查的目的不是延迟决策，而是减少盲动。

常见案例：
- 学习：不是“我学不会”，而是要查清楚卡在概念、练习量、反馈频率，还是考试策略。
- 工作：不是“领导不认可我”，而是要确认交付质量、沟通频率、岗位期待、可见度。
- 人际：不是“他不在乎我”，而是要区分行为事实、沟通差异和自己的安全感投射。

自我提问：
- 我亲眼确认的事实有哪些？
- 哪些只是我的猜测？
- 如果让我用证据说服一个旁观者，我手里有什么？
- 我还需要做哪一个最小调查？

行动模板：先不要下结论，给自己 48 小时补齐关键事实。补齐后再决定是继续、调整、沟通还是退出。

## 模型 2：主要矛盾模型

核心原理：任何复杂局面都有多个矛盾，但总有一个矛盾最支配全局。抓不住主要矛盾，人就会忙、累、乱，却不前进。

操作方法：
1. 把当前问题拆成所有矛盾：钱、时间、能力、关系、情绪、资源、方向。
2. 问：哪个矛盾不解决，其他努力都会被拖住？
3. 判断主要矛盾的主要方面：真正卡住的是自己、对方、环境、方法，还是阶段。
4. 把资源集中到这个点上。
5. 每 7-14 天复查一次，因为主要矛盾会变化。

误区：
- 把最痛的点当主要矛盾。最痛不一定最关键。
- 把最容易做的事当主要矛盾。容易只是降低焦虑，不一定改变局面。
- 同时解决所有问题。平均用力通常等于没有重点。

常见案例：
- 职业迷茫：表面是行业选择，主要矛盾可能是没有可验证能力样本。
- 关系冲突：表面是吵架，主要矛盾可能是双方对承诺、边界、未来预期不一致。
- 副业卡住：表面是流量少，主要矛盾可能是产品不清楚、目标人群不清楚、转化路径断裂。

自我提问：
- 如果只能解决一个问题，哪个问题解决后其他问题会明显变轻？
- 我现在做的事是在打主战场，还是在外围消耗？
- 这个问题的主导因素是我能控制的，还是暂时不能控制的？

行动模板：未来 7 天只围绕一个主矛盾行动。其他问题先记录，不分散主力。

## 模型 3：实践-认识循环模型

核心原理：正确认识不是在脑子里想出来的，而是在实践、反馈、修正中形成的。想清楚才行动，常常是内耗的高级形式。

操作方法：
1. 把问题转成实验：我想验证什么？
2. 设计最小行动：小到可以立刻开始，短到可以快速反馈。
3. 记录结果：有效、无效、意外发现。
4. 修正判断：保留被验证的，删除想象出来的。
5. 进入下一轮行动。

误区：
- 把一次失败当终局判断。
- 把计划当成果。
- 用“我还没准备好”保护自己不接受反馈。

常见案例：
- 写作：不要空想定位，连续发 10 篇，看哪类主题有真实反馈。
- 求职：不要只想适不适合，投递、面试、复盘，能力定位会变清楚。
- 亲密关系：不要反复脑补，对关键议题做一次诚实沟通，看对方是否愿意共同解决。

自我提问：
- 我现在缺的是思考，还是反馈？
- 哪个动作能在 3 天内给我新信息？
- 我害怕行动后看到什么结果？

行动模板：把困惑变成一个 7 天实验。实验失败也不是失败，而是获得情报。

## 模型 4：持久战模型

核心原理：许多真正重要的事都不是速胜战，而是持久战。错误的速胜幻想会制造焦虑、短期主义和自我否定。

操作方法：
1. 判断问题性质：这是短期任务，还是长期战役？
2. 拆成三个阶段：防御期、相持期、反攻期。
3. 防御期目标：活下来、稳住基本盘、停止失血。
4. 相持期目标：积累能力、建立节奏、寻找突破口。
5. 反攻期目标：集中资源、扩大成果、形成系统。

误区：
- 把长期问题当短期问题，期待立刻逆袭。
- 在防御期追求体面，在相持期失去耐心，在反攻期不敢加码。
- 把坚持理解成被动忍耐。持久战的核心是主动积累和等待时机。

常见案例：
- 学习一门硬技能：前期看不见成果，中期开始能做小作品，后期才形成竞争力。
- 走出低谷：先恢复作息和基本秩序，再重建能力和关系，最后才谈跃迁。
- 创业/副业：先验证需求，再稳定交付，再扩大流量和复购。

自我提问：
- 我现在处在哪个阶段？
- 这个阶段的正确目标是什么？
- 我是不是用反攻期标准惩罚防御期的自己？

行动模板：不要用一年后的目标压垮今天。先完成当前阶段的胜利条件。

## 模型 5：战略退却模型

核心原理：退却不是失败，盲目硬冲才可能导致真正失败。战略退却是为了保存力量、重新选择战场、等待反攻条件。

操作方法：
1. 判断是否继续失血：时间、钱、精力、尊严、机会成本是否持续恶化。
2. 区分撤退和逃避：撤退有新部署，逃避只有消失。
3. 设定底线：最多投入多少，再观察多久，达到什么信号就撤。
4. 撤到哪里：换方法、换岗位、换关系距离、换赛道、换节奏。
5. 保存火种：保留经验、资源、人脉、作品、复盘。

误区：
- 把不甘心当战略判断。
- 把沉没成本当继续理由。
- 把撤退做成情绪崩盘，而不是理性转移。

常见案例：
- 工作：一个岗位长期消耗且无成长，可以先转内部机会、补能力、准备外部选择。
- 关系：反复伤害且无法沟通，不是继续证明自己值得爱，而是恢复边界。
- 学习：方法长期无效，不是加倍痛苦，而是换路径、换老师、换材料。

自我提问：
- 我继续投入，是因为看见机会，还是因为不甘心？
- 如果我是旁观者，会建议自己继续还是撤？
- 撤退后我能保存什么力量？

行动模板：设置一个清晰止损线。到线就撤，撤退后立刻进入新部署。

## 模型 6：集中优势兵力模型

核心原理：弱者不能平均作战。普通人的时间、注意力和资源都有限，必须在关键点形成局部优势。

操作方法：
1. 列出所有正在消耗你的任务。
2. 标出能产生最大变化的 1-2 个突破口。
3. 停掉低收益、高消耗、只提供虚假忙碌感的动作。
4. 在突破口上设置连续投入周期：至少 14-30 天。
5. 用结果复盘是否继续加码。

误区：
- 把多线并进当努力。
- 把兴趣广泛当战略。
- 在每件事上都投入一点，最后没有一个形成战斗力。

常见案例：
- 学习：不要同时学五门课，先打穿一门能产生结果的核心技能。
- 工作：不要所有同事都讨好，先把最关键交付做好。
- 内容：不要每个平台都发，先找到一个能反馈的阵地。

自我提问：
- 我现在最大的分散是什么？
- 哪个点打穿后，会给我带来最大连锁变化？
- 我愿意为了它暂时放下什么？

行动模板：未来 30 天只打一场主仗。用少数关键动作换一个真实突破。

## 模型 7：阶段论模型

核心原理：不同阶段有不同任务。把后期任务提前，会焦虑；把前期任务拖延，会停滞。

操作方法：
1. 判断阶段：起步期、积累期、突破期、扩张期、调整期。
2. 给每个阶段定义胜利标准。
3. 起步期重在开始和反馈，不重在完美。
4. 积累期重在重复和系统，不重在惊艳。
5. 突破期重在集中资源和抓窗口。

误区：
- 起步期就要品牌感、规模化、稳定收入。
- 积累期天天换方向。
- 扩张期还用单兵作战方式。

常见案例：
- 新手写作：起步期先写完，积累期稳定发布，突破期打磨爆款主题。
- 职业转型：先了解行业，再做作品样本，再寻找机会。
- 健身：先建立习惯，再加训练量，再追求体型变化。

自我提问：
- 我现在到底在哪个阶段？
- 这个阶段最不该追求什么？
- 当前阶段完成的标志是什么？

行动模板：不要跨阶段焦虑。先把当前阶段该完成的动作做扎实。

## 模型 8：主动权模型

核心原理：人在问题中最危险的状态，是节奏、解释权和选择权都交给外界。主动权不是控制一切，而是在限制中找到可行动空间。

操作方法：
1. 区分可控、可影响、不可控。
2. 从可控部分拿回第一步行动。
3. 建立自己的信息渠道和判断标准。
4. 不等别人安排，主动提出方案、边界、时间表。
5. 保留替代选择。

误区：
- 把主动权理解成强势压人。
- 把不可控因素当借口。
- 等到被动局面恶化才行动。

常见案例：
- 职场：与其等评价，不如定期同步进展、争取资源、明确交付标准。
- 关系：与其猜对方，不如提出具体沟通和边界。
- 人生方向：与其等机会，不如做作品、见人、试项目。

自我提问：
- 这件事里我能控制的 20% 是什么？
- 我可以主动提出什么方案？
- 如果当前路径失败，我的替代路径是什么？

行动模板：今天就做一个能拿回主动权的小动作：发起沟通、提交样本、设定边界、建立备选。

## 模型 9：统一战线模型

核心原理：成事需要团结可以团结的力量。不要把所有差异都当敌对，也不要幻想只靠自己解决所有问题。

操作方法：
1. 明确共同目标。
2. 区分核心同盟、临时同盟、中立者、阻碍者。
3. 找到对方利益和你的目标之间的交集。
4. 先合作可合作部分，不强求价值观完全一致。
5. 对关键分歧保留边界。

误区：
- 要么完全信任，要么完全切割。
- 把合作当讨好。
- 忘记共同目标，只陷入情绪评价。

常见案例：
- 项目推进：先争取关键支持者，再影响中间人，不把全部精力耗在最反对的人身上。
- 家庭沟通：先找共同目标，如健康、经济、孩子成长，再谈具体做法。
- 创业：和互补的人合作，不要求对方与自己完全相同。

自我提问：
- 谁和我有共同利益？
- 我需要争取谁、影响谁、避开谁？
- 哪些分歧可以暂放，哪些底线不能让？

行动模板：先找到一个可以合作的人，一起完成一个小目标。不要孤军奋战。

## 模型 10：政策边界模型

核心原理：好的行动既要有方向，也要有边界。没有边界，容易过火；没有方向，容易摇摆。

操作方法：
1. 明确目标：我到底要达成什么？
2. 明确底线：什么不能做、不能牺牲、不能越界？
3. 明确弹性：哪些方法可以调整？
4. 明确尺度：做到什么程度算够？
5. 定期修正政策，不靠情绪临场发挥。

误区：
- 情绪上头时临时改变底线。
- 把原则僵化成不变通。
- 把灵活变成没有立场。

常见案例：
- 人际：可以理解别人，但不能长期接受羞辱和控制。
- 工作：可以加班救急，但不能让救急变成制度性剥削。
- 学习：可以调整方法，但不能放弃核心训练量。

自我提问：
- 我的目标是什么？
- 我的底线是什么？
- 哪些可以谈，哪些不能谈？

行动模板：先写下这件事的“三条政策”：目标、底线、可调整空间。

## 模型 11：领导方法模型

核心原理：领导不是发号施令，而是从群众中来、到群众中去：收集真实情况，形成判断，再回到实践中检验和推动。

操作方法：
1. 从一线收集问题、经验、阻力和建议。
2. 提炼共性，不被个别声音带偏。
3. 形成清楚方案。
4. 让执行者理解为什么做、怎么做、做到什么程度。
5. 收反馈，再修正。

误区：
- 只凭位置领导，不凭事实领导。
- 只下命令，不解释目标和方法。
- 只听上面，不看现场。

常见案例：
- 团队项目：先问执行人困难，再定计划，而不是闭门排期。
- 家庭事务：先了解每个人真实需求，再分工。
- 自我管理：把自己也当团队来管理，给不同目标安排清楚节奏。

自我提问：
- 我有没有真正了解执行现场？
- 我的方案能被执行者理解吗？
- 反馈机制在哪里？

行动模板：任何计划都要经过一线信息、方案形成、执行反馈三步。

## 模型 12：报告制度模型

核心原理：没有稳定反馈，系统就会失控。很多失败不是因为不努力，而是信息延迟、问题不上报、错误积累到不可收拾。

操作方法：
1. 建立固定报告周期：每日、每周、每阶段。
2. 报告只写四件事：做了什么、结果如何、遇到什么问题、下一步怎么改。
3. 对重大偏差立即报告，不等周期。
4. 让报告服务决策，不做形式主义。
5. 用报告积累复盘资料。

误区：
- 只报喜不报忧。
- 把报告写成长篇感想。
- 没有指标，只有情绪。

常见案例：
- 学习：每天记录学习时长、题量、错因、第二天调整。
- 工作：每周同步进度、风险、需要支持。
- 自我成长：每周做一次状态报告，避免问题暗中扩大。

自我提问：
- 我现在是否知道事情真实进展？
- 哪些问题被我拖着没有上报给自己或他人？
- 我的反馈周期是不是太长？

行动模板：建立一张周报表。每周用 20 分钟把行动、结果、问题、调整写清楚。

## 模型 13：批评与自我批评模型

核心原理：真正的反内耗不是否定自己，而是把错误从人格审判中解放出来，变成可分析、可修正的对象。

操作方法：
1. 先批评问题，不攻击人格。
2. 找事实：我具体哪里做错了？
3. 找原因：是认知、习惯、能力、情绪，还是环境？
4. 找改法：下一次如何避免？
5. 对他人批评也要指向事情和改进，而非羞辱。

误区：
- 把自我批评做成自我厌恶。
- 把不舒服都叫内耗，拒绝面对真实问题。
- 批评别人时只宣泄情绪。

常见案例：
- 拖延：不是“我废物”，而是任务太大、反馈太远、启动动作不清楚。
- 冲突：不是“我不值得被爱”，而是表达方式、防御反应、边界设置需要修正。
- 工作失误：不是完蛋，而是流程、检查、沟通节点出了问题。

自我提问：
- 我是在解决问题，还是在惩罚自己？
- 这个错误可以被拆成哪几个可改动作？
- 如果朋友犯同样错误，我会怎么帮他复盘？

行动模板：把“我不行”改写成“我在某个动作上缺少某种能力或机制”。

## 模型 14：人民内部矛盾模型

核心原理：不是所有冲突都是敌我冲突。多数生活、人际、团队问题，是目标、认知、利益、表达方式之间的内部矛盾。处理错性质，会把小问题打成大战争。

操作方法：
1. 判断对方是否有共同目标或基本善意。
2. 区分分歧、误会、利益冲突、伤害行为。
3. 对内部矛盾用沟通、协商、教育、制度、边界。
4. 对持续伤害、操控、暴力、欺骗，要升级边界和保护。
5. 不把一次分歧上升为人格定性。

误区：
- 把不同意见都当背叛。
- 为了和气回避真实矛盾。
- 对恶性伤害仍按普通沟通处理。

常见案例：
- 伴侣争吵：先判断是需求没被看见，还是价值观根本冲突。
- 同事协作：先看目标是否一致，再处理分工和责任。
- 家庭冲突：先把关心、控制、焦虑、利益分开。

自我提问：
- 这是普通分歧，还是持续伤害？
- 我和对方有没有共同目标？
- 该沟通、协商、设边界，还是退出？

行动模板：先定性冲突，再选方法。不要用战争方式解决普通分歧，也不要用温和幻想处理真实伤害。

## 模型 15：组织起来模型

核心原理：个体力量有限，组织能把分散资源变成系统能力。人生不是所有事都靠意志硬扛，而是要建立工具、流程、同伴和环境。

操作方法：
1. 找到重复性问题。
2. 把它变成流程、清单、模板、固定时间。
3. 寻找同伴、导师、社群或合作对象。
4. 把成果沉淀成系统，而不是每次重新开始。
5. 定期维护系统。

误区：
- 只靠热血，不建机制。
- 把求助当软弱。
- 系统太复杂，反而无法坚持。

常见案例：
- 学习：建立错题系统、复盘周期、练习计划。
- 工作：建立项目模板、沟通模板、风险清单。
- 生活：建立作息、预算、运动、饮食的低摩擦系统。

自我提问：
- 哪些痛苦是反复出现的？
- 我能不能用一个流程减少它？
- 谁或什么工具可以成为我的外部力量？

行动模板：不要只发誓改变。为一个高频问题建立一个最小系统。

## 模型 16：艰苦奋斗模型

核心原理：资源少不是失败理由，而是战略条件。艰苦奋斗不是苦情，而是在低资源下保持方向、纪律、创造力和长期积累。

操作方法：
1. 承认现实资源，不虚荣比较。
2. 保住基本盘：健康、现金流、核心能力、关键关系。
3. 用低成本方式试错。
4. 把困难变成训练：节制、耐心、判断力、执行力。
5. 小胜积累，逐渐扩大条件。

误区：
- 把吃苦当目的。
- 用贫乏审美合理化低质量。
- 一边资源少，一边做高消耗选择。

常见案例：
- 转行：先用业余时间做作品，不一开始裸辞。
- 创业：先低成本验证，不一开始铺大摊子。
- 低谷：先恢复秩序和现金流，再谈远大理想。

自我提问：
- 我现在最需要保住的基本盘是什么？
- 哪些开销、关系、任务在消耗我？
- 我能用什么低成本方式继续前进？

行动模板：在资源不足时，不追求体面开局，追求可持续推进。

## 16 个模型的选择表

| 问题状态 | 首选模型 |
|---|---|
| 不知道真实情况 | 调查研究 |
| 事情太乱 | 主要矛盾 |
| 想太多不行动 | 实践-认识循环 |
| 长期焦虑、想速成 | 持久战 |
| 继续投入很痛苦 | 战略退却 |
| 精力太分散 | 集中优势兵力 |
| 目标太远 | 阶段论 |
| 被别人牵着走 | 主动权 |
| 需要合作支持 | 统一战线 |
| 容易摇摆或过火 | 政策边界 |
| 带团队或推进项目 | 领导方法 |
| 反馈断裂 | 报告制度 |
| 自责内耗 | 批评与自我批评 |
| 人际冲突 | 人民内部矛盾 |
| 反复靠意志硬扛 | 组织起来 |
| 资源少、起点低 | 艰苦奋斗 |
